# student-life-os

A cloud-based student dashboard designed to help students manage tasks and expenses in one place.

## Purpose
This project is built to explore:
- Backend development
- Cloud infrastructure
- Platform-oriented system design

## Features (Planned)
- User authentication
- Task management
- Expense tracking
- Cloud deployment

## Tech Stack (Planned)
- Python (Backend)
- Flask / FastAPI
- Cloud (AWS or Azure)
